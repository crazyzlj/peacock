Supported platforms
-------------------

| host platform | target platform | compiler | architecture |
| ------------- | --------------- | -------- | ------------ |
| linux         | linux           | gcc, 4.9 | x86-64       |
| windows       | windows         | mingw    | x86-64       |

Other platforms may work but have not been tested.
